# ICONIX

A static website hosted on GitHub Pages that serves the [ICONIX Blog](https://iconix254.blogspot.com/).

## 🌐 Live Site

**GitHub Pages**: [https://ladybugsaga.github.io/ICONIX/](https://ladybugsaga.github.io/ICONIX/)

## 📁 Structure

```
ICONIX/
├── index.html        # Main page — embeds the ICONIX blog
├── 404.html          # Custom 404 error page
├── style.css         # Design system
├── sitemap.xml       # Sitemap for search engines
├── robots.txt        # Crawler directives
├── CNAME             # Custom domain (optional)
└── README.md         # This file
```

## 🔍 SEO & Google Search Console

This site is ready for Google Search Console verification:

1. **Sitemap** — Submit `https://ladybugsaga.github.io/ICONIX/sitemap.xml`
2. **robots.txt** — Configured to allow all crawlers
3. **Meta tags** — Open Graph and description tags included
4. **404 page** — Custom error page for better UX

## 🚀 Deployment

The site is deployed automatically via GitHub Pages from the `main` branch.
